# C++ Practice Files

This repository contains my C++ practice codes from when I first started learning programming.

## Files and Topics Covered

* **Search & Math:** Binary Search, Majority Element, Random Numbers.
* **Arrays:** Sorting, finding Max/Second Max, and shifting elements.
* **Data Structures:** Linked Lists, Vectors (STL), and Structs.
* **Small Projects:** Rock Paper Scissors game and a To-Do List app.

## How to Run
1. Open your terminal.
2. Compile: `g++ FileName.cxx`
3. Run: `./a.out` (on Linux/Mac) or `a.exe` (on Windows).
